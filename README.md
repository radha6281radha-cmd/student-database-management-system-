# Student Database Management System

## 📌 About the Project
This is a simple Student Database Management System developed using Python.  
I created this project to practice file handling and basic CRUD operations (Create, Read, Update, Delete).

The program stores student data like Roll Number, Name, Age, and Course in a text file and allows the user to manage records through a menu-driven interface.

## 🚀 Features
- Add new student details
- View all student records
- Search student by Roll Number
- Update existing student details
- Delete student record

## 🛠️ Technologies Used
- Python
- File Handling (Text File)

## ▶️ How It Works
- The data is stored in a file (`studentsinf3.txt`)
- Each record is saved in comma-separated format
- The program runs in a loop with a menu system

- Handling user input
- Implementing CRUD operations
